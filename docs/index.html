Pulse

A macOS menu-bar monitor for live
CPU, memory, GPU, disk, and network stats.

⬇︎ Download for macOS
Free · Apple Silicon · requires macOS 11+
CPU — overall %, E/P clusters, per-core bars, frequency
Memory — used / total, %, and swap
GPU — utilisation & power on Apple Silicon, no sudo
Network — live up / down throughput
Disk — used % per mounted volume
Power & thermals — CPU / GPU / ANE watts & temps

How to open the app

Pulse is signed ad-hoc, so macOS blocks it on first launch. Any one of these opens it, and you only do it once:

  1. Right-click (Control-click) Pulse.app in Finder → Open → Open.
  2. If that's blocked on newer macOS: System Settings → Privacy & Security → scroll down → "Open Anyway".
  3. Terminal fallback: /usr/bin/xattr -dr com.apple.quarantine /Applications/Pulse.app
